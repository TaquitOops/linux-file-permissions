# Linux File Permissions Portfolio Activity

## Overview

This repository contains a portfolio activity demonstrating how Linux commands are used
to inspect and manage file and directory permissions. The goal of the project is to ensure
that authorization levels are correctly applied in order to maintain system security.

This activity is part of a cybersecurity learning program and is intended to showcase
hands-on Linux and access control skills.

---

## Skills Demonstrated

- Managing Linux file and directory permissions
- Interpreting permission strings (`rwx`)
- Using `ls -la` to review permissions
- Using `chmod` to modify permissions
- Applying basic security principles
- Writing technical documentation in Markdown
